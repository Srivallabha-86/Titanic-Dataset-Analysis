# Titanic-Dataset-Analysis
Titanic Dataset using Python. Features comprehensive data cleaning, feature engineering, and a comparison of classification models (e.g., Logistic Regression) to predict passenger survival.

Project Overview: A brief intro about the Titanic disaster and the goal of the project—predicting survival based on passenger features like age, gender, and socio-economic class.

Dataset Dictionary: Defined the variables for clarity, such as Pclass (ticket class), SibSp (siblings/spouses), and Parch (parents/children).

Workflow Stages:Data Cleaning: Handled missing values (e.g., using the mean for Age or dropping the Cabin column).

Exploratory Data Analysis (EDA): Key insights, such as finding that women and children had significantly higher survival rates.

Feature Engineering: New variables created, such as "FamilySize" or extracting titles from names.

Modeling & Evaluation: Logistic regression classification algorithm and Evaluation metrics (Accuracy, Precision, Recall, F1-Score).

Technologies Used: Specific libraries like Pandas, NumPy, Matplotlib, Seaborn, and Scikit-learn.

Key findings: Class Advantage: Passengers in 1st Class had a much higher probability of survival than those in 3rd Class. Age Factor: Younger children generally had better survival rates due to "women and children first" policies.
